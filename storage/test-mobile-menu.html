🔍 Cách test Mobile Menu:

  1. Mở Developer Tools (F12)
  2. Chuyển sang mobile view hoặc thu nhỏ browser window
  3. Click vào hamburger button (☰)
  4. Kiểm tra Console để xem log messages
  5. Test đóng menu bằng cách click outside hoặc nhấn Escape

🐛 Debug Commands:

Mở Console và chạy các lệnh sau để debug:

// Kiểm tra mobile menu instance window.debugMobileMenu(); // Kiểm tra elements console.log('Menu Button:', document.querySelector('.mobile-menu-btn')); console.log('Nav Menu:', document.querySelector('.nav-menu')); // Test mở/đóng menu bằng code if (window.mobileMenuInstance) { window.mobileMenuInstance.toggle(); }
